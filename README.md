# DemonWave

A fast-paced, wave-based survival shooter inspired by classic arcade games and the DOOM universe. Fight endless hordes of demons, upgrade your character, and see how long you can survive!


## Description

DemonWave throws you into an arena against increasingly powerful demonic enemies. Survive as long as possible, earn score by killing monsters, and spend it between waves to become powerful enough for the next challenge.

**Core Gameplay:**
*   **Survival:** Face waves of six different demon types, each with unique stats and behaviors.
*   **Arsenal:** Unlock and switch between 5 different weapons (Pistol, Shotgun, Machine Gun, Sniper Rifle, Rocket Launcher) as you progress.
*   **Progression:** Spend earned score on permanent upgrades between waves.
*   **Difficulty Scaling:** Every 5 waves, demons receive random stat boosts, keeping the challenge intense.

## Features

- **6 Unique Demon Types:** Zombie, Imp, Cacodemon, Cyberdemon, Archvile, Hell Knight
- **5 Weapons:** Each with unique damage, fire rate, and behavior
- **Upgrade System:** Improve health, speed, damage, regeneration, and healing
- **Wave Progression:** Endless waves with increasing difficulty
- **Rich Audio:** Background music and unique sound effects for each weapon
- **Discord RPC:** Integration to show your gameplay status on Discord
- **Performance:** Smooth 60 FPS gameplay

## Installation

### Prerequisites
- Python 3.7+
- Pygame library
- (Optional) pypresence for Discord RPC

### Step-by-Step Installation

1. **Clone or download the project files**
   ```bash
   git clone https://github.com/vatson228/DemonWave/
   cd DemonWave
   pip install pygame psutil pypresence
   python doom.py```
